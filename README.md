# Hangman
Hangman adalah permainan klasik yang secara tradisional dimainkan dengan kertas dan pensil di mana satu pemain memikirkan sebuah kata dan pemain lain mencoba menebak apa itu. Setiap huruf yang ditebak dengan benar ditulis di semua tempat kemunculannya dalam kata.
